# Installation

## Stable release

To install geodemo_lee, run this command in your terminal:

```
pip install geodemo_lee
```

This is the preferred method to install geodemo_lee, as it will always install the most recent stable release.

If you don't have [pip](https://pip.pypa.io) installed, this [Python installation guide](http://docs.python-guide.org/en/latest/starting/installation/) can guide you through the process.

## From sources

To install geodemo_lee from sources, run this command in your terminal:

```
pip install git+https://github.com/LyMot/geodemo_lee
```
